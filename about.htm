This web site has been a chance to try out all kinds of things.

  • A bright punchy style that intends to deliver just a few points. This was inspired by Johnny Bunko, which summarizes six principles in a manga comic.
  • HSLA color specification.
  • CSS3 round borders.
  • Embedded fonts. Thank you, Font Squirrel.
  • A responsive CSS framework that uses media queries to provide formatting for any browser. Thank you, Dave Gamache, for Skeleton.
  • "Touch-friendly" navigation that doesn't use old-school links (good for tiny screens).
  • jQuery for hover effects, content-loading and in-page navigation.
  • My dog photos from two weeks ago. There is only one photo but I should be able to place a half-dozen more.
  • All the things I learned this decade. Still under construction... I mean, I am; and the pages are too.
